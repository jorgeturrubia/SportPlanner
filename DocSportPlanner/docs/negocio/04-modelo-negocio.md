## Resumen
El modelo de negocio propuesto es Freemium con asientos y planes escalados (Free, Personal, Pro, Director, Club).

### Planes y precios
- Free: 0€ — Cucurucho básico con límite de planificaciones y no acceso Marketplace
- Personal: 5,99€/mes — Entrenadores individuales, planificaciones ilimitadas
- Pro: 9,99€/mes — Acceso completo Marketplace
- Director: 15,99€/mes — Control multi-equipos + reportes
- Club: 19,99€/mes — CRM + licencias múltiples

- MVP con Free + Personal
- Beta con 10-20 entrenadores para validar funcionalidades core y precio
