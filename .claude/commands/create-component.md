# Create Component

Generate a new Angular component with Tailwind v4 styling and proper structure.

## Component Creation for: $ARGUMENTS

1. Generate component class with signals
2. Create template with Tailwind v4 classes
3. Add Hero Icons if needed
4. Include accessibility attributes
5. Create unit tests
6. Update module imports

## Structure
```
src/app/
└── features/
    └── [feature]/
        └── components/
            └── [component-name]/
                ├── [component-name].component.ts
                ├── [component-name].component.html
                ├── [component-name].component.spec.ts
                └── index.ts
```

Delegate to angular-dev agent.
