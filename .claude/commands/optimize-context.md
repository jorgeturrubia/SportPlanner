# Optimize Context

Trigger context optimization to reduce token usage and improve efficiency.

## Optimization Process

1. Analyze current context size
2. Identify redundant information
3. Compress or remove unnecessary details
4. Preserve critical requirements
5. Report token savings

## Triggers
- Context exceeds 35k tokens
- Before major development tasks
- When switching between features
- After completing milestones

## Expected Output
- Token count before/after
- Reduction percentage
- Preserved elements list
- Optimization technique used

Delegate to context-manager agent for execution.
