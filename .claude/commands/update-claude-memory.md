# Actualizador de Memoria Principal

Usa el claude-memory-manager para revisar y actualizar CLAUDE.md basándose en la documentación y agentes actuales.

## Cuándo usar:
- Después de crear nuevos agentes
- Después de modificar documentación del proyecto
- Después de cambios en la estructura del proyecto
- Cuando CLAUDE.md parece desactualizado

## Proceso:
1. El claude-memory-manager analizará el estado actual del sistema
2. Comparará CLAUDE.md actual con documentación reciente
3. Sugerirá actualizaciones necesarias
4. Preguntará antes de hacer cambios

## Uso:
```
/update-claude-memory
```

El agente mantendrá la información esencial y optimizará para tokens automáticamente.
